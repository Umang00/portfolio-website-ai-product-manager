# Project Image Generation Prompts

This document contains detailed prompts for generating poster images for all projects using Seedream or Nano Banana AI image generators.

## Prompt Structure Guidelines

Each prompt follows this structure:
- **Subject & Composition**: Main visual elements and layout
- **Style & Aesthetic**: Visual style, mood, and atmosphere
- **Color Palette**: Specific colors to use
- **Technical Details**: Lighting, perspective, and technical specifications
- **Key Visual Elements**: Important symbols, icons, or representations

## General Tips for Image Generation

1. **Aspect Ratio**: Use 16:9 (1920x1080) for all project posters for consistency
2. **Resolution**: Request high resolution (at least 1920x1080, preferably 4K)
3. **Style Consistency**: Maintain a modern, professional tech aesthetic across all images
4. **Color Harmony**: Use project-specific color palettes but maintain overall brand consistency
5. **Composition**: Center-focused with clear visual hierarchy
6. **Typography**: If text is needed, use modern, clean fonts
7. **Lighting**: Soft, professional lighting that highlights the main subject
8. **Negative Space**: Leave room for potential overlays or text additions

## Usage Instructions

1. Copy the full prompt for each project
2. Paste into Seedream or Nano Banana
3. Adjust aspect ratio to 16:9
4. Set resolution to high (1920x1080 or higher)
5. Generate and refine as needed
6. Use the alternative prompts if the main prompt is too complex
